Stable Bud is a lightweight desktop graphical user interface (GUI) built with Python that lets you generate images from text prompts using the Stable Diffusion machine learning model.

## Features
* **Simple GUI:** Clean, dark-mode interface powered by `customtkinter`.
* **Text-to-Image:** Enter a prompt and generate unique images.
* **Background Loading:** The heavy AI model loads in the background so the app UI doesn't freeze.
* **Auto-Save:** Generated images are automatically saved to your local directory as `generated.png`.

## Prerequisites
You need Python 3.8+ installed on your system. You will also need a Hugging Face account and an Access Token (Read).
